# Cinema-Forge

## Project Overview
Cinema-Forge is a comprehensive platform designed to streamline the process of managing cinema-related tasks through an intuitive user interface. The application provides tools for scheduling, ticketing, and managing cinema operations effectively.

## Features
- **User Management:** Allows users to register, log in, and manage their profiles.
- **Movie Listings:** Browse through a wide array of movies with details such as genre, duration, rating, and synopsis.
- **Ticket Booking:** Users can purchase tickets for their desired shows and receive electronic tickets.
- **Scheduler:** Managers can schedule movies and manage showtimes from a centralized dashboard.
- **Notifications:** Real-time alerts for upcoming shows, ticket availability, and promotions.
- **Payment Integration:** Support for various payment methods, including credit cards, PayPal, and more.
- **Admin Panel:** A dedicated space for administrators to manage movies, shows, and user accounts.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Payment Gateway:** Stripe
- **Deployment:** AWS for hosting and deployment

## Quick Start Guide
1. **Clone the repository:**  
   `git clone https://github.com/UBN606/Cinema-Forge.git`
2. **Navigate to the project directory:**  
   `cd Cinema-Forge`
3. **Install dependencies:**  
   For Frontend:  
   `cd client && npm install`  
   For Backend:  
   `cd server && npm install`
4. **Run the application:**  
   - Start the backend: `cd server && npm start`  
   - Start the frontend: `cd client && npm start`
5. **Access the application:**  
   Open your browser and go to `http://localhost:3000`
